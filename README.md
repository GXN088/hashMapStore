# hashMapStore

Recap - HashMap Operations


Создайте метод с именем processHashMap, который принимает один аргумент:

HashMap с именем products, где:
Ключи представляют названия продуктов (String).
Значения представляют цены (Double).
Операции, которые нужно выполнить
Найдите продукт с наивысшей ценой.
Если несколько продуктов имеют одинаковую наивысшую цену, верните любой из них.
Если HashMap пуст, верните пустую строку ("").
Отфильтруйте продукты, у которых цена больше 50.00, и сохраните их в новом HashMap.
Вычислите среднюю цену всех продуктов.
Если HashMap пуст, средняя должна быть 0.0.
Формат возвращаемого значения
Метод должен возвращать новый HashMap со следующими парами ключ-значение:

Ключ	Значение
"Highest"	Имя продукта с наивысшей ценой (String).
"Filtered"	HashMap (HashMap<String, Double>), содержащий только продукты с ценой больше 50.00.
"Average"	Средняя цена всех продуктов (Double).
